# Apollo Chrome extension (v1)

Right-click any page or selection → "Ask Apollo" → the Apollo web client opens
with the question prefilled. Nothing sends until you press Send there.

## What it can access — by design, almost nothing

- No host permissions, no content scripts, no `scripting` permission: the
  extension cannot read any page's content.
- It forwards only what Chrome's own APIs provide: the tab title/URL, and the
  text you highlighted yourself.
- The web client treats `?q=` as a prefill only and never auto-sends.

## Install (unpacked, until store submission)

1. Visit `chrome://extensions`, enable Developer mode.
2. "Load unpacked" → select this folder.
3. For a self-hosted web client, edit `APOLLO_WEB` in `background.js`.

`icon128.png` is generated by `apps/desktop/scripts/gen-assets.mjs`-style
tooling at package time; any 128×128 PNG works for an unpacked install.
